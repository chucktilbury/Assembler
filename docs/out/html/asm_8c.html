Detailed Description

This is a small driver that combines the cpp, gas, vm programs so that they can be run seamlessly. This is mostly about the command line interface.

Command parameters -h help text (created automatically) -r Preprocess, assemble, and run the file given on the command line. -d Preprocess, assemble, and debug the file given on the command line. (not implemented) -k Keep the intermediate files. Normally deleted. -v Set the verbosity level from nothing to ridiculous. -o Output file. (if -r && ! -k, then delete when complete) -p Where to place all output and intermediate files. If this is not provided, then the location of the input file is used. -D Preprocessor "defines". Multiple instances can be provided, and/or a list separated by a ':' character can be provided. -I Prepend an entry to the input search path for all inputs, not just preprocessor. -J Append an entry to the input search path for all inputs, not just preprocessor.
